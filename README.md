## 📖 Sobre o Projeto

Desenvolvi o **Industry Week SENAI** para ser mais do que uma simples landing page: é uma linha do tempo interativa e imersiva. A ideia é guiar o usuário pelas diferentes eras industriais — desde a fumaça das máquinas a vapor em 1760 até a conectividade da Indústria 4.0. 

Foquei muito em criar uma interface limpa, com tipografia elegante e transições que valorizem o conteúdo histórico do evento, garantindo uma experiência de navegação fluida em qualquer dispositivo.

O projeto foi desenvolvido para o evento Engrenagens do Tempo — referente à semana da indústria — da escola SESI em parceria com o SENAI, para uma apresentação sobre a Indústria 4.0.

Você pode conferir o projeto em produção aqui: 👉 **[Acessar Aplicação](https://brunoproencadev.github.io/industry-week-senai-prj/)**

## ✨ Funcionalidades Principais

* **Navegação Histórica:** Interface estruturada em capítulos ("1ª Era", "4 Eras", "Ind 4.0", "Legado") para guiar o usuário de forma cronológica.
* **Dark / Light Mode:** Alternância de temas (Claro e Escuro) disponível no cabeçalho para melhorar a acessibilidade e o conforto visual do usuário.
* **Design Imersivo:** Uso de tipografia elegante e imagens de fundo temáticas que reagem ao contexto de cada período histórico (ex: energia a carvão, metalurgia).
* **Responsividade Total:** A interface se adapta de forma fluida a dispositivos móveis, tablets e desktops de qualquer tamanho.

## 🚀 Como Executar o Projeto Localmente

Para rodar este projeto na sua máquina para testes ou modificações, siga os passos abaixo:

### Pré-requisitos

* Ter o [Git](https://git-scm.com/) instalado na sua máquina.
* Um editor de código, como o [VSCode](https://code.visualstudio.com/).

### Passo a Passo

1. **Clone o repositório:**
```bash
git clone [https://github.com/brunoproencadev/industry-week-senai-prj.git](https://github.com/brunoproencadev/industry-week-senai-prj.git)

```

#

2. **Acesse a pasta do projeto:**
```bash
cd industry-week-senai-prj

```


3. **Inicie a aplicação:**
Basta abrir o arquivo `index.html` diretamente no seu navegador ou utilizar a extensão **Live Server** do VSCode para uma experiência com *hot-reload*.
